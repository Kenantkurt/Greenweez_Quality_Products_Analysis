# Greenweez Quality Products Analysis

This repository contains the analysis of product refunds and net revenue for Greenweez. The project focuses on understanding why customers return products and how these refunds affect overall turnover.

## Project Overview
- Analyzed sales and refund data to calculate key metrics:
  - Total refund amounts (`ref_turnover`, `ref_ship`, `ref_coupon`)
  - Net turnover after refunds
  - Percentage contribution of each refund type
  - Turnover per order
- Identified the main reasons for returns and quantified their impact on revenue.

## Key Findings
- **Customer returns** had the largest impact, reducing turnover by approximately €1,648 (50.47%).  
- **Damaged products** and **wrong products** were the next most significant reasons.  
- Insights from this analysis help optimize revenue, reduce losses, and improve operational and financial decision-making.
